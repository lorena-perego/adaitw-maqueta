Somos ADA ITW

Somos una ONG que inició formal su labor en 2017. Nuestra misión es reducir la brecha de género en tecnología y esto lo hacemos a través del empoderamiento de las mujeres con cursos de programación. Creemos que la tecnología tiene un alto poder transformador y buscamos generar oportunidades para todas las mujeres alrededor del mundo.

Nuestros programas

Programador Front-End

+Info

El desarrollo Front-End maneja el lenguaje de programación JavaScript y consiste en optimizar recursos y diseñar cómo se ve la información con la que interactúa el usuario en un sitio web o app.

*GIT*HTML*CSS*JS*React*RD*

Programador Full-Stack

+Info

El desarrollo Back-End maneja el lenguaje de programación web Java y el objetivo en esta área se enfoca en cómo se va a guardar, organizar y proteger la información en un sitio web o en una app.

*GIT*POO*QA*Java*SQL*APIs*

Programador Python

+Info

En este curso incorporarás conocimientos claves para poder iniciarte en el mundo de la programación Back-End. Aprenderás a utilizar un IDE para escribir código en lenguaje Python y crear app.

*GIT*Python*SQL*
